Frontend წარმოადგენს ვებსაიტის მომხმარებლისთვის ხილულ ნაწილს, რომელიც იქმნება HTML, CSS და JavaScript-ის გამოყენებით.

What is Frontend?

Frontend არის ვებსაიტის ნაწილი, რომელსაც მომხმარებელი ხედავს და იყენებს. იგი იყენებს HTML-ს სტრუქტურისთვის, CSS-ს დიზაინისთვის და JavaScript-ს ფუნქციონალისთვის.

What is HTML?

HTML (HyperText Markup Language) არის ვებსაიტის ძირითადი სტრუქტურის ენად.

History

HTML შეიქმნა ტიმ ბერნერს-ლიმ მიერ 1991 წელს.

What is CSS?

CSS (Cascading Style Sheets) გამოიყენება ვებსაიტის დიზაინისა და ვიზუალის დასაპროექტებლად.

History

CSS-ის პირველი ვერსია გამოიცა 1996 წელს.

What is JavaScript?

JavaScript არის პროგრამირების ენა, რომელიც უზრუნველყოფს ვებსაიტზე დინამიურ ფუნქციონალობას.

History

JavaScript შეიქმნა ბრენდან აიხის მიერ 1995 წელს.
